Ejercicio 1: Diferentes métodos de salida por consola

Enunciado

En este primer ejercicio del tema, vamos a explorar las distintas formas que ofrece Java para mostrar información por pantalla. El objetivo es capturar 4 caracteres y utilizar un método de salida diferente para cada uno.

Requisitos:

  • Capturar una cadena de 4 caracteres por teclado.
  • write(): Imprimir el primer carácter.
  • print(): Imprimir el segundo carácter.
  • println(): Imprimir el tercer carácter.
  • printf(): Imprimir el cuarto carácter con formato.

Solución

Para trabajar con System.out.write(), es importante recordar que este método escribe bytes en el flujo de salida, por lo que suele ser necesario llamar a flush() para asegurar que el dato se visualice inmediatamente.

package Tema9.Ejercicios.ejercicio1;

import java.util.Scanner;

public class Ejercicio1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.println("Introduce 4 caracteres:");

        // Leemos la entrada completa como String
        String entradaCaracteres = sc.next();

        // Extraemos cada carácter por su posición
        char c1 = entradaCaracteres.charAt(0);
        char c2 = entradaCaracteres.charAt(1);
        char c3 = entradaCaracteres.charAt(2);
        char c4 = entradaCaracteres.charAt(3);

        // 1. Imprimir con write (requiere flush para forzar la salida)
        System.out.write(c1);
        System.out.flush();

        // 2. Imprimir con print (sin salto de línea)
        System.out.print(c2);

        // 3. Imprimir con println (añade un salto de línea al final)
        System.out.println(c3);

        // 4. Imprimir con printf (salida formateada usando %c para caracteres)
        System.out.printf("%c", c4);
        
        sc.close();
    }
}

Diferencias entre métodos

  • write: Es un método de bajo nivel que maneja bytes. No añade saltos de línea ni formatea datos automáticamente.
  • print: Convierte el dato a texto y lo imprime sin añadir nada al final.
  • println: Similar a print, pero añade automáticamente el carácter de nueva línea \n.
  • printf: Permite interpolar variables en una cadena mediante especificadores de formato (como %d para enteros o %s para strings).
