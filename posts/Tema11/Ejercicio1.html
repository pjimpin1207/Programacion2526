Ejercicio 1: Creación de Ventanas y Controles Interactivos con AWT

Enunciado

Entramos de lleno en el desarrollo de aplicaciones visuales utilizando **AWT (Abstract Window Toolkit)**, la API nativa de Java para la gestión de ventanas. El objetivo de este primer ejercicio práctico es construir un contenedor gráfico básico provisto de un campo de inserción de texto y dos botones interactivos independientes.

Elementos requeridos:

  • Diseñar una ventana principal interactiva con título descriptivo.
  • Añadir un botón funcional etiquetado como "Salir".
  • Añadir un segundo botón funcional etiquetado como "Limpiar".
  • Incorporar un cuadro de texto editable (TextField) posicionado debajo de las directivas de control anteriores.

Solución

Para construir el contenedor visual recurrimos a la clase factoría de alto nivel Frame. Con el fin de evitar que los componentes se solapen o colapsen el lienzo completo de la ventana, aplicamos un gestor de distribución secuencial simple denominado FlowLayout.

package Tema11.Ejercicios.ejercicio1;

import java.awt.*;

public class Ejercicio1 {
    public static void main(String[] args) {
        // 1. Inicializar la ventana contenedora principal (Frame) con su título
        Frame frame = new Frame("Hola Mundo");
        frame.setSize(300, 200); // Dimensionamos ancho y alto en píxeles
        frame.setVisible(true);   // Forzamos al sistema operativo a renderizar la ventana

        // Configurar el Administrador de Distribución (Layout Manager)
        // FlowLayout organiza los componentes en fila de izquierda a derecha y salta de línea si no caben
        frame.setLayout(new FlowLayout());

        // 2. Creación e inserción del botón "Salir"
        Button boton = new Button("Salir");
        frame.add(boton);

        // 3. Creación e inserción del botón "Limpiar"
        Button botonLimpiar = new Button("Limpiar");
        frame.add(botonLimpiar);

        // 4. Creación e inserción del campo editable TextField de una sola línea
        TextField texto = new TextField("Escribe algo..");
        frame.add(texto);
    }
}
