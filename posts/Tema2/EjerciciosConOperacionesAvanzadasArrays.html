Ejercicio 16: B√∫squeda Binaria

Enunciado

Implementa una funci√≥n de b√∫squeda binaria. Recuerda que este algoritmo solo funciona si el array est√° ordenado.

Soluci√≥n

# Ejercicio 16 - B√∫squeda Binaria

def busquedaBinaria(numeros, eleccion):
    izq = 0
    der = len(numeros) - 1
    medio = 0

    while izq <= der:
        medio = (izq + der) // 2

        if numeros[medio] == eleccion:
            return medio
        elif numeros[medio] < eleccion:
            izq = medio + 1
        else:
            der = medio - 1
    return -1
