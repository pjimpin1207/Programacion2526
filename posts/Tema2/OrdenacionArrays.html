Ejercicio 17: Ordenación Alfabética Ignorando Mayúsculas

Enunciado

Disponemos de una lista de tareas desordenadas. Queremos obtener un informe ordenado alfabéticamente, pero sin que las mayúsculas o minúsculas afecten al orden.

Ejemplo: “comprar café” debe aparecer antes que “Llamar al cliente”

Solución

# Ejercicio 17 - Ordenación sin distinguir mayúsculas/minúsculas

tareasDia = ["Revosar email", "llamar cliente", "comprar cafe",
             "actualizar reporte", "planificar reunion"]

# Crear lista ordenada alfabéticamente ignorando mayúsculas
informeOrdenado = sorted(tareasDia, key=str.lower)

# Imprimir lista ordenada temporal
print("Informe ordenado alfabéticamente sin distinguir mayúsculas:")
print(informeOrdenado)

# Imprimir lista original
print("Lista original:")
print(tareasDia)

Con sorted(lista, key=str.lower) conseguimos una ordenación estable y exacta, ignorando mayúsculas.

Ejercicio 18: Comparación entre Búsqueda Secuencial y Binaria

Enunciado

Debes realizar los siguientes pasos:

  • Crear un array de números desordenados.
  • Buscar un número usando búsqueda secuencial.
  • Buscar el mismo número usando búsqueda binaria.
  • Comparar ambos resultados.
  • Ordenar el array y repetir ambos algoritmos.
  • Comparar de nuevo los resultados.

Solución

# Ejercicio 18 - Comparar búsqueda secuencial y búsqueda binaria

# Importamos las funciones del ejercicio 15 y 16
import Ejercicio16
import Ejercicio15_2

# Crear un array desordenado
arrayDesordenado = [3, 15, 2, 21, 15, 66, 50, 43, 51]
print("Array original desordenado:", arrayDesordenado)

# Número a buscar
buscar = int(input("Introduce el número que deseas buscar: "))

# Buscar en el array desordenado
indiceSecuencial = Ejercicio15_2.busquedaSecuencial(arrayDesordenado, buscar)
indiceBinaria = Ejercicio16.busquedaBinaria(arrayDesordenado, buscar)

print("\nResultados en el array desordenado:")
print("Búsqueda secuencial:", indiceSecuencial)
print("Búsqueda binaria:", indiceBinaria)

# Ordenar el array
arrayDesordenado.sort()
print("\nArray ordenado:", arrayDesordenado)

# Buscar de nuevo en array ordenado
indiceSecuencialOrdenado = Ejercicio15_2.busquedaSecuencial(arrayDesordenado, buscar)
indiceBinariaOrdenada = Ejercicio16.busquedaBinaria(arrayDesordenado, buscar)

print("\nResultados en el array ordenado:")
print("Búsqueda secuencial:", indiceSecuencialOrdenado)
print("Búsqueda binaria:", indiceBinariaOrdenada)

# Explicación final:
# - La búsqueda secuencial funciona igual en un array ordenado o desordenado.
# - La búsqueda binaria SOLO funciona correctamente si el array está ordenado.

Conclusión

  • Búsqueda Secuencial: funciona siempre, pero es más lenta.
  • Búsqueda Binaria: rapidísima, pero requiere un array ordenado.
