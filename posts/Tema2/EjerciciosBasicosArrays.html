Ejercicio 2: Suma de los valores de un array

Crea un array de 5 n√∫meros y usa tres tipos de bucles para calcular la suma: for, for range y while.

Soluci√≥n

# for
numeros = [3, 7, 2, 8, 5]
suma = 0

for elemento in numeros:
    suma = suma + elemento

print("La suma de los n√∫meros con for es:", suma)

# for i in range
numeros = [3, 7, 2, 8, 5]
suma2 = 0

for i in range(len(numeros)):
    suma2 += numeros[i]

print("La suma de for in range es:", suma2)

# while
suma3 = 0
i = 0
while i < len(numeros):
    suma3 = suma3 + numeros[i]
    i = i + 1

print("La suma con while es:", suma3)

Ejercicio 3: ¬øQu√© est√° pasando y c√≥mo corregirlo?

C√≥digo dado:

a = [1, 2, 3, 4]
for i in range(3):
    print(a[i])

Explicaci√≥n

El programa solo muestra los primeros 3 elementos porque range(3) genera 0, 1 y 2. Para recorrer todo el array, debe usarse len(a).

Correcci√≥n:

for i in range(len(a)):
    print(a[i])

Ejercicio 6: Mostrar n√∫meros en orden inverso

Solicitar cu√°ntos n√∫meros quiere introducir el usuario, guardarlos y mostrarlos al rev√©s.

Soluci√≥n

# Solicitar cu√°ntos n√∫meros desea introducir
cantidad = int(input("¬øCu√°ntos n√∫meros enteros deseas introducir?: "))

numeros = []

# Rellenar lista

    numero = int(input("Introduce el n√∫mero entero " + str(i + 1) + ": "))
    numeros.append(numero)

print("\nLos n√∫meros en orden inverso son:")
for n in reversed(numeros):
    print(n)

# 2. Sin reversed()
print("\nLos n√∫meros en orden inverso son:")
for i in range(len(numeros) - 1, -1, -1):
    print(numeros[i])

# 3. Con while
print("\nLos n√∫meros en orden inverso son:")
i = len(numeros) - 1
while i >= 0:
    print(numeros[i])
    i -= 1

Ejercicio 7: Funci√≥n maximo(t)

Soluci√≥n

# Dise√±ar una funci√≥n que devuelva el valor m√°ximo de una lista

def maximo(t):
    maximo_valor = t[0]
    for elemento in t[1:]:
        if elemento > maximo_valor:
            maximo_valor = elemento
    return maximo_valor
