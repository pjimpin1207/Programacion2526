Ejercicio 9: Interfaz Comparable y Orden Natural

Enunciado

En este ejercicio aprenderemos a definir el orden natural de nuestros propios objetos. Para que una colección sepa cómo ordenar elementos automáticamente, la clase debe implementar un contrato específico.

Objetivos y Tareas:

  • Escoger la colección adecuada para almacenar elementos sin repetidos y en orden natural.
  • Implementar la interfaz Comparable en la clase Persona para ordenar por nombre.
  • Asegurar la consistencia sobrescribiendo equals() y hashCode() (dos personas son iguales si tienen mismo nombre y apellidos).
  • Migrar los datos a una colección que respete el orden de inserción.

Solución

La solución requiere modificar la clase Persona para que implemente Comparable<Persona> y defina la lógica de comparación.

1. Clase Persona (Implementando el contrato)

package Tema8.Ejercicios.ejercicio9;

import java.util.Objects;

// Implementamos Comparable para definir el orden natural
public class Persona implements Comparable<Persona> {
    private String nombre;
    private String apellidos;
    private int edad;

    public Persona(String nombre, String apellidos, int edad) {
        this.nombre = nombre;
        this.apellidos = apellidos;
        this.edad = edad;
    }

    public String getNombre() { return nombre; }
    public String getApellidos() { return apellidos; }
    public int getEdad() { return edad; }

    // Definimos el orden natural: Alfabético por nombre
    @Override
    public int compareTo(Persona otra) {
        return this.nombre.compareTo(otra.nombre);
    }

    // Consistencia: Dos personas son iguales si coinciden nombre y apellidos
    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;
        Persona persona = (Persona) o;
        return Objects.equals(nombre, persona.nombre) &&
                Objects.equals(apellidos, persona.apellidos);
    }

    @Override
    public int hashCode() {
        return Objects.hash(nombre, apellidos);
    }

    @Override
    public String toString() {
        return nombre + " " + apellidos + " (" + edad + ")";
    }
}

2. Clase Principal (Main)

Utilizamos TreeSet para el orden natural y LinkedHashSet para el orden de inserción.

package Tema8.Ejercicios.ejercicio9;

import java.util.*;

public class Main {
    public static void main(String[] args) {
        // TreeSet gestiona automáticamente el orden natural (Comparable)
        Set<Persona> listaNatural = new TreeSet<>();

        // Añadir personas (incluyendo un intento de duplicado lógico)
        listaNatural.add(new Persona("Fran", "Gomez", 30));
        listaNatural.add(new Persona("Ana", "Perez", 25));
        listaNatural.add(new Persona("Luis", "Lopez", 40));
        listaNatural.add(new Persona("Ana", "Perez", 80)); // No se añadirá por ser "igual" a la anterior

        System.out.println("Colección ordenada por nombre (natural): " + listaNatural);

        // Comprobación de consistencia entre equals y hashCode
        Persona p1 = new Persona("Ana", "Perez", 25);
        Persona p2 = new Persona("Ana", "Perez", 80); 

        System.out.println("¿Son p1 y p2 la misma persona (equals)?: " + p1.equals(p2));
        System.out.println("HashCode p1: " + p1.hashCode());
        System.out.println("HashCode p2: " + p2.hashCode());

        // Copiar a una colección que mantenga el orden de inserción
        Set<Persona> listaInsercion = new LinkedHashSet<>(listaNatural);
        System.out.println("Colección por orden de inserción: " + listaInsercion);
    }
}

Conceptos Técnicos Clave

1. Orden Natural: Es el que define la clase por defecto al implementar Comparable. En este caso, hemos elegido el nombre.

2. Consistencia: Si dos objetos son iguales según equals(), deben devolver 0 en compareTo() y tener el mismo hashCode(). Esto es vital para que colecciones como TreeSet o HashSet funcionen correctamente.

3. LinkedHashSet: Lo hemos usado para "congelar" el orden actual y que no cambie si añadimos más elementos, respetando la cronología.
