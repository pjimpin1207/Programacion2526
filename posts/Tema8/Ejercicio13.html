Ejercicio 13: Map

Enunciado

En este ejercicio dejamos atrás las colecciones secuenciales para trabajar con Mapas. La interfaz Map representa un diccionario de datos basado en parejas de Clave - Valor.

Contexto:

Imagina que eres el desarrollador de una frutería. El dueño necesita consultar existencias de forma instantánea. Usar una lista obligaría a buscar elemento por elemento, pero con un Map obtendremos una búsqueda directa y eficiente.

Solución

Implementamos la clase InventarioFruteria utilizando un HashMap, donde la clave es el nombre de la fruta (String) y el valor es su stock (Integer).

package Tema8.Ejercicios.ejercicio13;

import java.util.HashMap;
import java.util.Iterator;
import java.util.Map;

public class InventarioFruteria {
    public static void main(String[] args) {
        // 1. Crear el mapa (Clave: String, Valor: Integer)
        Map<String, Integer> inventario = new HashMap<>();

        // 2. Carga de datos inicial
        inventario.put("Manzanas", 50);
        inventario.put("Plátanos", 20);
        inventario.put("Naranjas", 35);
        inventario.put("Peras", 12);

        // 3. Consulta directa por clave
        System.out.println("Cantidad de naranjas: " + inventario.get("Naranjas"));

        // 4. Actualización (Sobrescribir el valor de una clave existente)
        inventario.put("Manzanas", 100);
        System.out.println("Cantidad de manzanas actualizadas: " + inventario.get("Manzanas"));

        // 5. Verificación de existencia
        if (inventario.containsKey("Mangos")) {
            System.out.println("Hay mangos en el inventario");
        } else {
            System.out.println("No se han encontrado mangos");
        }

        // 6. Recorrido usando keySet y un Iterador
        System.out.println("\n--- Inventario Completo ---");
        Iterator<String> itFrutas = inventario.keySet().iterator();
        while (itFrutas.hasNext()) {
            String fruta = itFrutas.next();
            Integer cantidad = inventario.get(fruta);
            System.out.println(fruta + ": " + cantidad);
        }
    }
}

Preguntas de Reflexión

1. Al actualizar "Manzanas", ¿aparecen dos entradas o se sobrescribe?
Se sobrescribe. En un HashMap las claves son únicas. Al usar .put() con una clave que ya existe, el mapa simplemente reemplaza el valor antiguo por el nuevo.

2. ¿Qué pasaría si usas un int como clave?
Daría un error de compilación. Las colecciones en Java solo trabajan con objetos. Deberíamos usar la clase envoltorio Integer.

3. ¿Por qué es más eficiente un Map que un ArrayList aquí?
Por la velocidad de búsqueda. En un ArrayList $(O(n))$, el programa debe recorrer todos los elementos hasta encontrar el correcto. En un Map $(O(1))$, el programa va directamente a la posición de la clave gracias a su función hash.
