Ejercicio 12: ListIterator - Potencia en la Iteración

Enunciado

En este ejercicio exploraremos ListIterator, una versión más potente del iterador tradicional que solo está disponible para clases que implementan List. A diferencia de un iterador normal, este nos permite movernos en ambas direcciones y modificar la lista mientras la recorremos.

Requisitos:

  • Inicializar un ArrayList<Integer> con 10 números.
  • Recorrer la lista hacia adelante mostrando los índices y multiplicando por 2 los valores mayores de 50.
  • Recorrer la lista hacia atrás e insertar el número 25 después de cualquier valor menor que 30.
  • Utilizar métodos como nextIndex(), previousIndex(), set() y add().

Solución

El uso de ListIterator es ideal cuando necesitamos realizar ediciones complejas sobre una lista sin los problemas de concurrencia que daría un bucle for tradicional.

package Tema8.Ejercicios.ejercicio12;

import java.util.ArrayList;
import java.util.List;
import java.util.ListIterator;

public class Ejercicio12 {
    public static void main(String[] args) {
        // 1. Inicializar la lista con 10 números
        List<Integer> numeros = new ArrayList<>();
        for (int i = 10; i <= 100; i += 10) {
            numeros.add(i);
        }

        System.out.println("Lista inicial: " + numeros);

        // 2. Crear el ListIterator
        ListIterator<Integer> it = numeros.listIterator();

        // 3. Recorrido hacia adelante con modificación (set)
        System.out.println("\n--- Recorriendo hacia adelante ---");
        while (it.hasNext()) {
            int indice = it.nextIndex();
            Integer valor = it.next();

            System.out.println("Índice " + indice + ": " + valor);

            // Si el valor es mayor que 50, lo actualizamos en la lista
            if (valor > 50) {
                it.set(valor * 2); 
            }
        }
        System.out.println("Lista tras multiplicar > 50 por 2: " + numeros);

        // 4. Recorrido hacia atrás con inserción (add)
        // El iterador ya está al final, así que podemos empezar a retroceder
        System.out.println("\n--- Recorriendo hacia atrás ---");
        while (it.hasPrevious()) {
            int indice = it.previousIndex();
            Integer valor = it.previous();

            System.out.println("Índice " + indice + ": " + valor);

            // Si el valor es menor que 30, añadimos el número 25
            if (valor < 30) {
                it.add(25); // Se añade en la posición actual
                it.previous(); // Reposicionamos para no entrar en bucle infinito
            }
        }
        System.out.println("Lista final: " + numeros);
    }
}

Diferencias entre Iterator y ListIterator

  • Dirección: Iterator solo va hacia adelante. ListIterator es bidireccional (previous()).
  • Modificación: Iterator solo permite eliminar (remove()). ListIterator permite sustituir (set()) y añadir (add()).
  • Índices: ListIterator puede decirnos en qué posición está en todo momento con nextIndex().
  • Ámbito: Iterator sirve para cualquier Collection. ListIterator es exclusivo de List.
