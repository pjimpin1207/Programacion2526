Ejercicio 27 – Múltiplos de 7

Publicado por pjimpin1207

Enunciado

Escribe un programa que muestre por pantalla todos los múltiplos de 7 menores que 100.

Explicación del ejercicio

📌 ¿Qué es un múltiplo?

Un número es múltiplo de 7 cuando puede escribirse como:

7 × n, siendo n un número entero.

Los primeros múltiplos serían: 7, 14, 21, 28, ...

📌 Generación de múltiplos con un bucle for

El bucle comienza en 7 y aumenta de 7 en 7:

for (int i = 7; i < 100; i += 7)

Esto garantiza que cada valor recorrido sea un múltiplo de 7, sin necesidad de usar el operador módulo %.

Solución en Java


public class Ejercicio27 {
    public static void main(String[] args) {

        for (int i = 7; i < 100; i += 7) {
            System.out.println(i);
        }
    }
}
