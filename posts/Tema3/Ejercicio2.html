Enunciado

Escribe un programa que pida la edad al usuario y muestre la edad que tendrá el año que viene.

Explicación del ejercicio

Este ejercicio combina entrada de datos con una operación aritmética muy simple. El usuario introduce su edad, y el programa calcula su edad futura sumándole 1.

Pasos que se realizan:

  • Crear un objeto Scanner para leer la entrada.
  • Pedir la edad al usuario.
  • Leer el valor con nextInt().
  • Sumar 1 para obtener la edad del año siguiente.
  • Mostrar el resultado.

Solución en Java


import java.util.Scanner;

public class Ejercicio2 {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce tu edad: ");

        int edad = sc.nextInt();          // Leer edad del usuario
        int edadSiguiente = edad + 1;     // Calcular edad futura

        System.out.println("El año que viene vas a tener: " 
                           + edadSiguiente + " años");
    }
}

Resultado esperado


Introduce tu edad: 19
El año que viene vas a tener: 20 años
