Ejercicio 30 – Triángulo de Asteriscos

Enunciado

Pedir al usuario un número entero n y dibujar un triángulo rectángulo de n elementos de lado utilizando asteriscos (*).

Ejemplo para n = 4:


* * * *
* * *
* *
*

Conceptos que se practican

  • Uso de bucles for anidados.
  • Control de filas y columnas.
  • Impresión de patrones de texto.

Solución en Java


import java.util.Scanner;

public class Ejercicio30 {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        System.out.print("Introduce un número n: ");
        int n = sc.nextInt();

        // Bucle para controlar las filas
        for (int i = n; i >= 1; i--) {

            // Bucle para imprimir i asteriscos en cada fila
            for (int j = 1; j <= i; j++) {
                System.out.print("* ");
            }
            System.out.println();
        }
    }
}
