Ejercicio 26 – Aplicación ECO

Enunciado

Crear una aplicación llamada Eco qué pida al usuario un número entero y muestre en pantalla la palabra:

Eco...

repetida tantas veces como indiqué el número introducido.

Explicación del ejercicio

📌 Uso del bucle for

El ejercicio utiliza un bucle for porqué conocemos de antemano cuántas veces debe repetirse la acción.

for (int i = 0; i < numeroIntroducido; i++)

En cada iteración del bucle se imprime la palabra "Eco...".

Solución en Java


import java.util.Scanner;

public class Ejercicio26 {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce un número: ");
        int numeroIntroducido = sc.nextInt();

        for (int i = 0; i < numeroIntroducido; i++) {
            System.out.println("Eco...");
        }
    }
}
