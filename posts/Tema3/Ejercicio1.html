Ejercicio 1 – Entrada y salida de datos en Java

Enunciado

Diseñar un programa qué pida un número al usuario por teclado y a continuación lo muestre por consola.

Explicación del ejercicio

Para leer un número en Java utilizamos la clase Scanner, qué permite capturar datos introducidos por el usuario a través del teclado. Después, simplemente mostramos el número usando System.out.println().

Pasos qué realiza el programa:

  • Crea un objeto Scanner.
  • Pide al usuario qué introduzca un número.
  • Lee el número con nextInt().
  • Lo muestra por consola.
  • Cierra el escáner.

Solución en Java


import java.util.Scanner;

public class Ejercicio1 {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);      // Crear escáner para leer por teclado
        System.out.print("Introduzca un número: ");
        int numero = sc.nextInt();                 // Leer número entero

        System.out.println("El número es: " + numero);  // Mostrar número por consola
    }
}
