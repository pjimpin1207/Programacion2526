Ejercicio 5 – Media aritmética de dos notas

Publicado por pjimpin1207

Enunciado

Crear una aplicación que calcule la media aritmética de dos notas enteras. Se debe tener en cuenta que la media puede contener decimales.

Explicación del ejercicio

Aunque las notas pueden ser enteras, la media puede ser decimal, por lo que debemos trabajar con el tipo double. La fórmula es:

media = (nota1 + nota2) / 2

En este ejercicio no se piden los valores al usuario, sino que se asignan directamente para demostrar el cálculo de la media.

Conceptos aplicados:

  • Tipos de datos decimales (double)
  • Operaciones aritméticas
  • Impresión por consola

Solución en Java


public class Ejercicio5 {
    public static void main(String[] args) {

        double nota1 = 8.77;
        double nota2 = 7.67;

        double media = (nota1 + nota2) / 2;

        System.out.println("La media de las notas es: " + media);
    }
}

Resultado esperado


La media de las notas es: 8.22

(El valor puede variar ligeramente según redondeos.)
