Tema 5 – Ejercicio 6

Clases del JDK: Object, String, Math y Random

🧠 Enunciado

Busca las siguientes clases dentro de tu librería JDK:

  • Object
  • String
  • Math
  • java.util.Random

📚 Clases analizadas

  • Object

    Es la clase raíz de la jerarqué de clases en Java. Todas las clases heredan de Object de forma directa o indirecta.

  • String

    Clase utilizada para trabajar con cadenas de texto. Es inmutable y pertenece al paqué java.lang.

  • Math

    Proporciona métodos estáticos para realizar operaciones matemáticas como potencias, raíces cuadradas, redondeos o valores absolutos.

  • Random

    Pertenece al paqué java.util y permite generar números aleatorios de distintos tipos.
